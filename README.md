
         PAGE REPLACEMENT ALGORITHM SIMULATOR - HOW TO USE
         


[Step 1] Open Terminal
         
[Step 2] Type: streamlit run app.py
       
[Step 3] Press Enter

[Step 4] Wait for "Local URL: http://localhost:8501"

[Step 5] Click the link OR copy to browser

[Step 6] Browser opens with simulator

[Step 7] Look at LEFT SIDEBAR

[Step 8] Select Algorithm (example: FIFO)

[Step 9] Type Number of frames (example: 3)

[Step 10] Type Reference string (example: 1 2 3 4)

[Step 11] Check "Enable Animation" (optional)

[Step 12] Click "RUN SIMULATION"

[Step 13] Watch the animation and results!
